Detailed Description

La classe EditorGame gère l'ensemble des paramètres pouvant être passés au programme Pacman.

Author
Valentin D.d.G.
Version
1.0

La classe gère un ensemble de niveaux dont les paramètres sont via une UI. EditorGame peut s'instancier avec un document XML et s'exporter de la même façon afin de sauvegarder les données.

Constructor & Destructor Documentation

EditorGame::EditorGame ( )

Constructeur de base de EditorGame.

Crée une instance de EditorGame vide qui ne dispose d'aucun niveau ni d'aucun paramètres. Lors de cette instanciation, le gestionnaire de texture est créé afin d'être utilisé par l'ensemble des niveaux.
