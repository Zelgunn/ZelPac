Detailed Description

Gestionnaire de textures.

Author
Valentin D.d.G.
Version
1.0

Ce gestionnaire de textures permet de faire le lien entre le nom complet d'un fichier, son nom en court et son index dans le gestionnaire. Le gestionnaire par défaut modifie la taille des textures.
