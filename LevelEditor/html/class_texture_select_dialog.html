Detailed Description

Classe de fenêtre de sélection de texture.

Author
Valentin D.d.G.
Version
1.0

En fonction du gestionnaire de texture passé en paramètre, la fenêtre affiche les textures que le gestionnaire contient et les affiche en forme de liste. Lorsque la fenêtre se ferme, il est possible de récupérer l'index de la dernière texture sélectionnée via la fonction selectedTextureIndex.
