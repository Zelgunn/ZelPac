Detailed Description

Classe imitant une position à virgule avec deux entiers.

Author
Valentin D.d.G.
Version
1.0

DoublePosition utilise deux couples d'entiers afin de simuler un couple de réels afin d'éviter les pertes de précisions dûes à la représentation binaire.

void DoublePosition::increaseX ( int  pace)

Augmente la coordonnée X réelle.

Parameters
[in]pacePas d'augmentation de la partie réelle (en X).

Augmente de pace la partie réelle selon X.
Si la partie réelle excède MAX_FLOAT_POS alors elle devient nulle et la partie entière est augmentée de 1.
De même, si la partie réelle devient inférieure à 0, la partie entière diminue de 1 et la partie réelle est mise à MAX_FLOAT_POS moins le pas.
Si la partie réelle s'approche de 0, elle devient alors nulle.
