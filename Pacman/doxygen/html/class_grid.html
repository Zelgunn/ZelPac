Detailed Description

Classe de grille de Pacman.

Author
Valentin D.d.G.
Version
1.0

La Grille s'occupe d'attribuer à chaque case une texture et un indicateur de collision.

La classe génère une image de base qui sera mise à jour le minimum possible (une case par frame en théorie) afin d'être utilisée par d'autres classes ultérieurement. La grille s'occupe aussi de baliser la maison des fantômes.
