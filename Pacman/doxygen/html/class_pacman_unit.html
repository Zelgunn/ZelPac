Detailed Description

Classe de l'unité Pacman.

Author
Valentin D.d.G.
Version
1.0

Cette classe héritée de la classe Unit permet d'ajouter quelques fonctionnalités uniques au Pacman (comme l'enregistrement d'un coup en avance) ainsi qu'une aide pour le dessin du Pacman.

Surcharge du constructeur (avec grille) pour utiliser XML.

Parameters
gridLa grille sur laquelle se trouve Pacman.
elemDonnées XML à charger lors de la contruction de l'objet.
parent

Construit un pacman (unité) attaché à une grille et dont les paramètres sont chargés à partir d'une balise XML.

  • speed : Vitesse par défaut de l'unité.
  • sspeed : Vitesse lorsque les fantômes sont apeurés.
  • Autres : Voir DoublePosition(QDomElement elem).

Exemple : <Player ix="14" iy="23" speed="77" sspeed = "91" fx = "half"/>

void PacmanUnit::setDirection ( int  direction)

Tente de modifier la direction du pacman.

Parameters
directionNouvelle direction.

Essaie de modifier la direction actuelle du joueur par le paramètre. Si le joueur ne peut pas effectuer ce mouvement immédiatement, la direction est stockée et sera prise dès que possible. Si une direction est déjà stockée afin d'être utilisée dès que possible, elle est écrasée. Si la nouvelle direction est égale à la direction actuelle, la prochaine direction est écrasée (afin d'annuler la commande).
