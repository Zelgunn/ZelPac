Detailed Description

Widget affichant l'ensemble du jeu.

Author
Valentin D.d.G.
Version
1.0

Grâce à l'utilisation d'un PaintEvent, cette classe fille de QWidget permet d'afficher tous les éléments graphiques du jeu en plein écran.

Cette classe permet aussi de recevoir les commandes utilisateurs venant du clavier ou d'une manette Arduino Uno.

void Pacscreen::onKeyboardInput ( int  direction)

Fonction de traitement des entrées de flèches directionnelles.

Parameters
directionTouche appuyée.

Lorsque cette fonction est appelée, si le jeu est totalement arrêté (victoire ou défaite), alors il redémarre. Dans le cas où celui ci est uniquement en pause (perte de vie, fin de niveau) alors il reprend. Sinon, le signal de la touche est propagé afin de changer la direction du joueur.
