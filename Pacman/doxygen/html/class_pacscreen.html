Detailed Description

Widget affichant l'ensemble du jeu.

Author
Valentin D.d.G.
Version
1.0

Grâce à l'utilisation d'un PaintEvent, cette classe fille de QWidget permet d'afficher tous les éléments graphiques du jeu en plein écran.

Cette classe permet aussi de recevoir les commandes utilisateurs venant du clavier ou d'une manette Arduino Uno.
